# tienda-

proyecto de tienda para programación 2 Universidad



\# MercaEZ



Proyecto universitario desarrollado por estudiantes de la Universidad de Córdoba.



\## Descripción



MercaEZ es una tienda virtual orientada a la gestión y compra de productos en línea.



\## Funcionalidades



\- Login de usuarios

\- Panel administrador

\- Catálogo de productos

\- Carrito de compras

\- Historial de compras

\- Lista de deseos



\## Tecnologías



\- Lenguaje: Java (JDK 17+)

\- Entorno de Desarrollo: NetBeans IDE

\- Interfaz Gráfica: Java Swing (JFrame)

\- Base de Datos: MySQL

\- Diseño UI/UX: Figma

\- Control de Versiones: Git y GitHub



\## Estructura del proyecto



\- tienda-virtual/: proyecto Java NetBeans

\- figma/: prototipos y recursos visuales

\- docs/: documentación
